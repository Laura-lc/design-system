Cards

These cards are built using flex box and expand to the size of the columns they are added to. Hopefully, in theory, they should work with grid layouts as well.

Simple Example

No styles added

CSS

              
                  /* Generic cards */
    
                  .ds-c-card {
                    display: flex;
                    flex-direction: column;
                  }

HTML

              
              <!-- Generic Card -->
    
                <div class="ds-c-card">
                    <article>
    
                      <!--content goes here-->
    
                    </article>
                </div>

Judicial branch

The judicial branch is comprised of the Supreme Court and other federal courts.

  • Legislative – Makes laws (Congress)
  • Executive – Carries out laws (President, Vice President, Cabinet)
  • Judicial – Evaluates laws (Supreme Court and other courts)

Legislative Branch

The legislative Branch enacts legislation, confirms or rejects Presidential appointments, and has the authority to declare war.

This branch includes Congress (the Senate and House of Representatives) and several agencies that provide support services to Congress. American citizens have the right to vote for Senators and Representatives through free, confidential ballots.

Branches of Government

Each branch of government can change acts of the other branches as follows:

  • The President can veto laws passed by Congress.
  • Congress confirms or rejects the President's appointments and can remove the President from office in exceptional circumstances.

Styled Examples

To get different styled cards, apply utility styles to the <article> tag. 

                
                  <!-- Card 1 utility classes -->
                  <div class="ds-c-card">
                      <article class="ds-u-padding--4 ds-u-fill--background-inverse ds-u-color--white ds-u-shadow--lg">
                      </article>
                  </div>
                  
                  <!-- Card 2 utility classes -->

                  <div class="ds-c-card">
                      <article class="ds-u-padding--3 ds-u-fill--white ds-u-border--2 ds-u-border--dark">
                      </article>
                  </div>

                  <!-- Card 3 utility classes -->

                  <div class="ds-c-card">
                      <article class="ds-u-padding--4 ds-u-fill--gold ds-u-radius--md ds-u-shadow--md">
                      </article>
                  </div>

Cards w/ images

Add a <figure> tag with an <img> above the <article> tag. No need to reformat the image, clipping has been added to make sure the card will stay in the correct aspect ratio.

CSS

              
                  /* Include image in a card */
                  
                  .ds-c-card figure {
                    display: flex;
                    justify-content: center;
                    align-items: center;
                    overflow: hidden;
                    margin: 0;
                  }
                  
                  .ds-c-card figure img {
                    flex-shrink: 0;
                    min-width: 100%;
                    min-height: 100%;
                    max-width: inherit;
                  }

HTML

              
              
                <!-- Card w/ image -->
    
                <div class="ds-c-card">
                    <figure>
                      <img src="https://via.placeholder.com/800x450)" />
                    </figure>
                    <article>
    
                      <!--content goes here-->
    
                    </article>
                </div>

Column Images

Apply .ds-u-flex-direction--row to .ds-c-card get a column layout for cards and images.

HTML

              

                <!-- Horizontal Card w/ image -->
    
                <div class="ds-c-card ds-u-flex-direction--row">
                    <figure>
                      <img src="https://via.placeholder.com/800x450)" />
                    </figure>
                    <article class="ds-l-col--6">
    
                      <!--content goes here-->
    
                    </article>
                </div>

Equal Heights

If you want all cards to be equal heights

CSS

              
                  
                  /* Equal height cards */
                  .ds-c-cards {
                    -webkit-box-align: stretch !important;
                    -ms-flex-align: stretch !important;
                    align-items: stretch !important;
                  }
                  
                  .ds-c-cards .ds-c-card {
                    height: 100%;
                  }
                  
                  .ds-c-cards .ds-c-card article {
                    height: 100%;
                  }

HTML

              
                <!-- Card equal heights -->
    
                <div class="ds-l-row ds-c-cards">
                    <div class="ds-l-col">
                      <div class="ds-c-card">
                        <article>
                        </article>
                      </div>
                    </div>
                    <div class="ds-l-col">
                      <div class="ds-c-card">
                        <article>
                        </article>
                      </div>
                    </div>
                    <div class="ds-l-col">
                      <div class="ds-c-card">
                        <article>
                        </article>
                      </div>
                    </div>
                  </div>

Utility Classes

Here are some extra utility classes added so cards can look right

Flex direction

This could be added the flex utility. THis is used to get the cards w/ images to format in a horizontal direction.

        .ds-u-flex-direction--row {
          flex-direction: row;
        }

Button utilities

Possibly in the button section. If you would like to center buttons or stretch them the entire lenght of the card you would need to apply one of these classes.

      .ds-u--center {
        margin: 0 auto;
        display: block;
      }
      
      .ds-u--stretch {
        width: 100%;
      }

Hr dividers

Hr's are good ways to divide content without having to add a new row or extra classes.

      /* HR */
      .ds-u-transparent {
        background-color: transparent;
        border-color: transparent;
      }
      
      .ds-u-space--2 {
        margin: 16px 0;
      }
      
      .ds-u-space--3 {
        margin: 24px 0;
      }
      
      .ds-u-space--4 {
        margin: 32px 0;
      }

Shadows

Medicare.gov uses shadows on their cards.

      /* Shadow styles used on https://tailwindcss.com/docs/box-shadow/ */

      .ds-u-shadow {
        box-shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06);
      }
      
      .ds-u-shadow--md {
        box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);
      }
      
      .ds-u-shadow--lg {
        box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05);
      }
      
      .ds-u-shadow--xl {
        box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1), 0 10px 10px -5px rgba(0, 0, 0, 0.04);
      }
      
      .ds-u-shadow--2xl {
        box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.25);
      }
      
      .ds-u-shadow--inner {
        box-shadow: inset 0 2px 4px 0 rgba(0, 0, 0, 0.06);
      }
      
      .ds-u-shadow--outline {
        box-shadow: 0 0 0 3px rgba(66, 153, 225, 0.5);
      }
      
      .ds-u-shadow--none {
        box-shadow: none;
      }

Border Radius

Currently there is only one border radius size. THinking about this more, maybe it would be better to just have one border radius size due to adding options adds complications to development and design.

      }
      .ds-u-radius--md {
        border-radius: 6px;
      }
      .ds-u-radius--lg {

      }

        border-radius: 12px;
